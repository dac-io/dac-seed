## Public

Generated directory that should be served to the user.

Conforms to the following structure:

- *index.html*: Autogenerated entry point.
- *views/*
    - *component.html*
    - *app.html*
- *scripts/*
    - *controllers.js*
    - *directives.js*
    - *filters.js*
    - *services.js*
    - *app.js*
- *styles/*
    - *component.scss*
    - *app.css*
